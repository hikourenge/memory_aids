# README

## サービス概要
利用者が“オリジナルクイズカード”を作成・配布できる Web アプリ。画像とテキストを自由に設定し、自由にオリジナルのカードを作成できるだけでなく、他者が作ったカードを検索・利用することで様々な知識を楽しみながら学ぶことを目指します。

## このサービスへの思い・作りたい理由
暗記学習に苦手意識を持つ学習者が多く、講義や教科ごとに最適な教材は自作が必要で暗記カードなどのアプリはあふれています。一方、教師などがが簡単に自作して学習者へ配布し、反復学習できるツールが不足しています。特に単純な暗記カードにならず、クイズやかるた形式といった形で、楽しく反復できるような工夫を施したツールが必要だと感じています。  
最初は問題文が順に表示されるクイズ形式で楽しみつつ学べるツールを目指しつつ、いずれは「かるた」のように 「読み→瞬発的に取る」 体験を学習に応用し、学びを遊びへ転換することで定着を狙えるようなツールにしていきたいと考えています。

## ユーザー層について
主に小学生から大学生までの学生、およびその教育者（教師・塾講師・保護者）  
  
学生の学習は暗記が必要なものが多い一方で、暗記は単調になりがちで嫌いと考えている学習者が一定数います。  
また、教育者がオリジナルのコンテンツを自作できるサービスは少なく、教育者が作成し配布できるようにすることで、暗記学習をより簡単なものにするため上記のようなユーザ層をターゲットとして考えています。


## サービス利用のイメージ
**想定ユーザー**
* 出題者（教師・塾講師・保護者も含む）
  * 教育者は定期テスト/受験範囲に合わせた重要語句のクイズカードを作成
  * その他利用者全員は任意でクイズカードを作成可能  

上記によってオリジナルの暗記クイズカードを教育者が作成可能。  
作成したカードを共有リンクを用いて学習者へ配布し学習を促すことができる
* 学習者
  * 作成されたクイズデッキで自主学習
  * 教育者から指定のカードを使用して学習・演習
  * 解答時間でのランキングや、CPU対戦で“勝負”のドキドキを楽しみながら反復  

クイズやゲーム性を持った機能を使うことで、学習範囲の暗記を楽しみながら進めることができる

## ユーザーの獲得について
現状は私が教育関係に勤務しているので、まずは現職場の教員などに利用をお願いする。  
安定して機能が利用できるようであれば、徐々にSNSを用いて暗記を楽しめるツールとしてアピールする。

## サービスの差別化ポイント・推しポイント
暗記カード作りのサービスは多数あるが、あくまで自主学習がメインで自分でカードを作成することがメインになっているものが多いと感じています。  
またカルタのようなゲーム性のあるものもあるが、セットが決められており教育者が自分で作ることができないものも多いです。  
教育者が自由に作成し、学習者へ配布できる点、多数の参加者が自作のカードを作り、公開しあうことで色々な知識を自由に学習しやすくなる点が他のサービスとの差別化のポイントになると考えています。  

## 機能候補
### MVP段階
**出題者**
1. 新規登録/ログイン → デッキ一覧
2. デッキ作成（タイトル/説明/ジャンルのタグ付け/公開設定/共有リンク自動発行）
3. カード追加（問題/答え/補足/画像プレビュー）
4. デッキ詳細で「共有リンク」をコピー → 学習者へ配布

**学習者**
1. 共有リンク or デッキ一覧（公開デッキのみ）から学習する内容を選択
2. タイトル、タグによる検索機能
3. モード選択（MVPは「暗記練習」のみ）
4. 1問ずつ表示 → 「解答を見る」→「正誤を自己申告」→ 次へ
5. 結果画面：正答率・所要時間の表示、デッキ一覧へ戻る

### 拡張機能（本リリースまでに目指したいこと）
* 学習者が間違えた問題だけを集めて再度復習用の画面を表示する機能
* 出題者が問題をcsvでインポートできる機能
* 問題に対するお気に入り機能
* 得点率と所要時間によりポイントを計算してランキング表示の機能  

その他、カルタのようなゲーム性を持たせた機能もいずれは実装したいと空想しています。
